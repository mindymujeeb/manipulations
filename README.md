
# Manipulations

Website useful for performing various String operations as mentioned below

## Features

- Uppercase
- Lowercase
- Capitalise
- Remove Extra Spaces
- Copy

## Run Locally

link of website

```bash
  https://manipulations.netlify.app
```

Go to the project directory

```bash
  cd manipulations
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack (Client Side)

- React
- CSS
- Javascript
